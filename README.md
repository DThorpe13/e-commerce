# e-commerce

### [Live Site](https://worldofmusic-commerce.netlify.app/)

## Introduction

A fully functional eCommerce application using React.js and Commerce.js, as well as Stripe for card transactions. 

## Technologies used

React.js
React Hooks
Commerce.js
Material.UI
Stripe(pay portal)
Netlify(deployment)
